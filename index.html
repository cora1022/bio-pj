수면제 조합과
복용 반응 시뮬레이터

위에서는 작용기전 비중을 조절해 하나의 가상 수면제를 구성하고, 아래에서는 그 조합이 상대 복용량에 따라 얼마나 안정적으로 보이는지 시각화합니다. 실제 제조법이나 실제 용량이 아니라 교육용 상대 모델입니다.

모델 개요

E(x) = Emax xγ / (EC50γ + xγ)
EBliss = 1 - Π(1 - Epathway)
Ptox = 1 / (1 + e-(a + bB))
Stability = 100 × (1 - Ptox)

상단 슬라이더는 조합 비중이며, 하단 슬라이더는 그 조합의 정규화 노출량입니다. `RU`는 relative unit, `AU`는 alcohol unit이며, 보고서에서는 `C/EC50`에 대응하는 무차원 노출지표로 해석할 수 있습니다.

주의

이 시뮬레이터는 실제 처방, 실제 제조, 실제 복용 용량을 뜻하지 않습니다. 기전 비중과 정규화 노출이 안정성에 미치는 방향성을 시각화하는 용도입니다.

실제 약물 제조법, 실제 복용량, 치명량 계산은 제공하지 않습니다. 이 페이지의 `%`, `RU`, `AU`는 모두 교육용 정규화 단위이며, 논문식 PK/PD 개념 설명을 위한 무차원 축척입니다.

수학적 모델과 논문 근거

아래 수식은 실제 약동학·약력학 연구에서 널리 쓰이는 형태를 바탕으로 단순화한 교육용 모델입니다. 이 페이지의 노출량은 실제 mg가 아니라 `C/EC50`형 정규화 축척으로 해석합니다. 각 링크는 해당 수식 계열의 근거 논문 또는 공식 라벨로 연결됩니다.

1. 경로별 농도-효과: Hill / sigmoid Emax E_i(x) = Emax_i * x^gamma_i / (EC50_i^gamma_i + x^gamma_i)

각 작용기전의 효과를 포화형 농도-효과 곡선으로 두었습니다. GABA, 멜라토닌, 오렉신, 히스타민 축 모두 이 기본형 위에서 기울기와 EC50만 다르게 두는 방식입니다. 본 시뮬레이터의 입력축은 실제 농도 대신 `정규화 농도 x = C/EC50`에 대응합니다. 논문 링크

2. 같은 경로 내 성분 합산: Loewe형 dose-addition surrogate x_GABA = d * (1.35f_barb + 1.00f_benzo + 0.90f_z) E_GABA = Emax_GABA * x_GABA^gamma / (EC50^gamma + x_GABA^gamma)

같은 경로에서 작동하는 성분은 먼저 하나의 “등가 노출”로 합산한 뒤 한 개의 Hill 곡선에 통과시켰습니다. 이는 같은 경로 내 dose-addition을 가정하는 Loewe/isobologram 계열의 직관을 따른 단순화입니다. 논문 링크

3. 다른 경로 간 병합: Bliss independence E_total = 1 - (1 - E_GABA)(1 - E_MEL)(1 - E_ORX)(1 - E_H1)

서로 다른 경로의 효과는 독립적인 확률적 기여로 보고 Bliss independence 형태로 합쳤습니다. 이 방식은 경로가 다른 약물 조합의 기대효과를 계산할 때 널리 쓰입니다. 논문 링크

4. 병용 위험항과 최종 안정성 B = 1.15E_GABA(1 + 0.75f_barb) + 0.22E_MEL + 0.42E_ORX + 0.60E_H1 + 0.95E_A + lambda * E_A * (E_GABA + E_H1 + 0.4E_ORX) + 0.85V P_tox = 1 / (1 + exp(-(a + bB))) Stability = 100 * (1 - P_tox)

최종 위험은 logistic 변환으로 확률형 지표로 맵핑했습니다. 알코올과 다른 CNS depressant의 병용 경고는 실제 라벨에서도 반복적으로 제시되므로, 상호작용항을 별도로 두었습니다. 졸피뎀 FDA 라벨 / 수보렉산트 FDA 라벨
